This is a todo web app without JavaScript, i.e. no async calls. It uses the action attribute of the form tag to send data to a go HTTP/2 server through a POST request. The server, in turn, is responsible for the MongoDB connection, serving static assets and generating html files from templates using the html/template package.

This app was mainly made in order to learn more about the net/http package, the html/template package and the MongoDB driver package.

The source code, as well as instructions on how to run this app on your machine, is available on GitHub under an MIT license.
